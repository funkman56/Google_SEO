step 1  Generate a token for password  (https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/creating-a-personal-access-token)

step 2  Open terminal (DOS)  

             $ git clone https://github.com/username/repo.git
                 
                 Username:  your_username
                 Password:  your_token

step 3  Git command 

            $ git init
            $ git status
            $ git add .
            $ git commit -m "first commit"
            $ git push (git push -f)

            $ git help (git -h)
